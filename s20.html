一、软件资讯

 

1.1、GitHub

全球最大开源托管平台。

是开发者们“殿堂般”的存在，很多软件的发布布，都会选择在这里。

看它的 GitHub Trends 就知道当下最火的应用是什么，比如最近大火了什么 AI 应用，在这里都能第一时间找到。

1.2、52Pojie

 

如果说 GitHub 是北乔峰，那么 52Pojie 就是南慕容。

虽然实力上比不了 GitHub，但它却是中国活跃度最高的软件交流社区。

它不但养活了一大批“黑科技”公众号，而且在这里你能找到很多独属于“中国网络环境”的软件。

比如“视频号下载”这样的软件，就全部出自 52Pojie。地址：

 

1.3、小众软件

一个成立了很久时间的软件分享网站。

在这里你能找到各式各样新鲜的软件，还有它们的使用介绍。

而且，小众软件还新开了一个“发现分享”的频道，在这里你能看到软件开发者们的交流探讨。

 

1.4、少数派

可以理解为软件界的“小红书”。

相比其它网站仅仅是简单的使用介绍，少数派里的文章更多的是“经验分享”。

但也因为少数派社区的“苹果用户”占比过高，被调侃“没有 Mac 谈什么生产力”？
