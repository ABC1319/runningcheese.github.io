一、以图搜图网站

 

▼ 通用搜索 简介
🌐 Google Images 国外第一，可能是最好用的。👍
🌐 Yandex Images 国外第二，谷歌的最佳补充，免魔法，用有“老司机引擎”的称号。👍
🌐 Bing Images 国外第三，效果不如谷歌，差异化也不如 Yandex。
🌐 TinEye Images 国外老牌，有自己独立的数据库，但效果不尽人意。
🌐 百度识图 国内第一，但效果不尽人意。
🌐 搜狗识图 国内第二，但效果不尽人意。
🌐 360 识图 国内第三，但效果不尽人意。
▼ 聚合搜索 简介
🌐 ImgOps 可同时搜索多个搜索引擎。👍
🌐 Reverse Image 可同时搜索 Google、Bing、Yandex、Baidu 四个引擎。
🌐 Reverse Image 可同时搜索 Google、Bing、Yandex、Baidu 四个引擎。
▼ 图库搜索 简介
🌐 immerse 基于 AI 驱动，内容主要来自 Pixabay、Pexels、Unsplash 等免费图库。👍
🌐 Pinterest 有自己独立数据库，可用扩展 Search By Image 来实现。
🌐 Wallhaven 有自己独立数据库，全球最大的壁纸网站。
🌐 Freepik 有自己独立数据库，国外的知名素材网站。
🌐 搜图神器 有自己独立数据库，国内的知名素材网站。
🌐 FindThatMeme 有自己独立数据库，国外表情包搜索网站。
🌐 逗比拯救世界 有自己独立数据库，国外表情包搜索网站。
▼ AI 搜索 简介
🌐 Lexica 以图搜图，但 AI 图库，可用于搜索类似图片。👍
🌐 Enterpix 以图搜图，但 AI 图库，可用于搜索类似图片。
🌐 BeenTrained？ 以图搜图，但 AI 图库，可用于确定你的照片是否被用于 AI 训练。
🌐 Image Identify 图片识别，上传图片后会自动识别图片内容。
▼ 动漫搜索 简介
🌐 IQDB.org 二次元搜索，包括动画、动漫、游戏、壁纸图片等内容。
🌐 3D IQDB 三次元搜索。
🌐 SauceNA 二次元搜索，内容主要来自 Pixiv。
🌐 Ascii2d 二次元搜索，内容主要来自 Pixiv、Twitter 等。
🌐 Trace.moe 动漫搜索，提供一张截图即可获得动漫来源，还有截图片段所在位置。
▼ 商品搜索 简介
🌐 淘宝 电脑浏览器上已无法使用，需要使用手机 APP。
🌐 京东 需要登录才能使用。
🌐 阿里1688 无需登录。
🔷 AliPrice卖家助手 可同时聚合搜索多家电商平台的图库。
▼ 人脸识别 简介
🌐 PimEyes 人脸搜索，号称最先进的人脸反向搜索引擎，但需要付费。
🌐 Xslist 人脸识别，老司机搜索引擎，主要向面日本小姐姐。
🌐 Pornstarbyface 人脸识别，老司机搜索引擎，主要向面欧美小姐姐。
▼ 其它搜索 简介
🌐 NSII 鸟类识别，野生鸟类图像识别。
🌐 iPlant 植物智 植物识别，植物物种图像识别。
🌐 Plant.id 植物识别，植物物种图像识别，国外网站。
🌐 EUIPO 专利搜索，欧洲专利局。
🌐 WIPO 专利搜索，欧洲专利局。
